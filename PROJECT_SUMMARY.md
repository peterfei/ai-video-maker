#### 2. AI语义理解引擎

```python
# 关键词提取

- 中英文转换
```

#### 3. 智能素材管理器
```python
# 混合策略
- 本地素材库优先
- 在线API补充
- 自动分类索引
- 持续积累优化
```

### 工作流程

```
脚本内容
   ↓
语义分析 → 提取关键词
   ↓
本地查找 → 找到？使用 ✓
   ↓ 没找到
在线搜索 → 下载素材
   ↓
保存到库 → 下次可用
   ↓
返回素材 → 生成视频
```

### 配置选项

## 📞 快速开始

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 生成第一个视频
python3 generate.py --script examples/sample_script.txt

# 3. 启用自动素材（编辑配置文件）
# config/default_config.yaml -> auto_materials.enabled: true

# 4. 生成带自动素材的视频
python3 generate.py --script examples/auto_material_demo.txt

# 5. 批量生成
python3 generate.py --batch data/scripts
```
